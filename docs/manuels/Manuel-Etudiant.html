1. Introduction et Accès au Portail

1.1 Présentation du portail étudiant

Le portail étudiant de Joda Company est votre espace personnel en ligne qui vous permet de suivre en temps réel l'avancement de votre candidature pour une bourse d'études en Chine. Depuis ce portail, vous pouvez :

  • Consulter le statut de votre dossier de candidature
  • Télécharger vos documents requis
  • Suivre l'état de vos paiements
  • Déclarer vos paiements effectués
  • Recevoir des notifications importantes de l'équipe
  • Consulter et modifier vos informations personnelles

1.2 Accès à l'application

L'application Joda Company est disponible en deux versions :

VersionDescriptionAccès
Application WebAccessible depuis tout navigateur InternetURL fournie par votre agent
Application DesktopInstallée sur votre ordinateur Windows, fonctionne hors-ligneIcône sur le bureau / menu démarrer
Note : Vos identifiants de connexion (nom d'utilisateur et mot de passe) vous sont transmis par votre agent Joda Company lors de votre inscription. Conservez-les précieusement.

1.3 Navigateurs compatibles

  • Google Chrome (recommandé)
  • Mozilla Firefox
  • Microsoft Edge
  • Safari (Mac/iPad)

1.4 Connexion au portail

Étapes de connexion :
  1. Ouvrez l'application (navigateur ou desktop)
  2. Saisissez votre nom d'utilisateur dans le champ prévu
  3. Saisissez votre mot de passe
  4. Cliquez sur le bouton « Se connecter »
  5. Vous serez automatiquement redirigé vers votre portail étudiant
Important : Si vous ne parvenez pas à vous connecter, contactez immédiatement votre agent Joda Company. Ne partagez jamais vos identifiants avec une autre personne.

2. Première Connexion et Changement de Mot de Passe

2.1 Obligation de changer votre mot de passe

Lors de votre première connexion au portail, une fenêtre de changement de mot de passe s'affiche automatiquement. Vous devez obligatoirement définir un nouveau mot de passe avant de pouvoir accéder à votre espace. Cette mesure de sécurité protège votre compte.

2.2 Procédure de changement de mot de passe

Étapes :
  1. La fenêtre « Changement de mot de passe requis » apparaît automatiquement
  2. Saisissez votre mot de passe actuel (le mot de passe temporaire fourni par votre agent)
  3. Saisissez votre nouveau mot de passe
  4. Confirmez le nouveau mot de passe dans le champ de confirmation
  5. Cliquez sur « Enregistrer »
  6. Votre portail s'ouvre automatiquement

2.3 Règles pour le mot de passe

  • Minimum 8 caractères
  • Incluez des lettres majuscules et minuscules
  • Incluez au moins un chiffre
  • Évitez les mots de passe évidents (prénom, date de naissance)
Conseil : Notez votre nouveau mot de passe dans un endroit sûr. En cas d'oubli, contactez votre agent pour une réinitialisation.

5. Gestion des Documents

5.1 Présentation

La section Documents vous permet d'envoyer les pièces justificatives requises pour votre candidature. Votre agent vous indiquera quels documents sont nécessaires selon votre profil et l'université cible.

5.2 Types de documents acceptés

Type de documentFormats acceptés
PasseportPDF, JPG, PNG
Relevés de notesPDF, JPG, PNG
DiplômesPDF, JPG, PNG
Photo d'identitéJPG, PNG
Acte de naissancePDF, JPG, PNG
Lettre de motivationPDF
Certificat médicalPDF, JPG, PNG
Preuve de paiementPDF, JPG, PNG
Taille maximale : Chaque fichier ne doit pas dépasser 10 MB. Les images sont automatiquement compressées. Les PDF volumineux doivent être compressés avant envoi.

5.3 Comment envoyer un document

Étapes :
  1. Allez dans la section « Documents »
  2. Identifiez le type de document à soumettre
  3. Cliquez sur « Choisir un fichier » ou glissez-déposez le fichier
  4. Vérifiez que le fichier s'affiche dans le champ
  5. Cliquez sur « Envoyer »
  6. Un message de confirmation apparaît si l'envoi réussit
  7. Optionnellement, cliquez sur « Envoyer à l'équipe » pour notifier votre agent

5.4 Statut de vos documents

Chaque document soumis affiche un statut :

StatutSignificationAction
En attente Document soumis, en attente de vérification par l'équipe Aucune action requise
Validé Document accepté par l'équipe Aucune action requise
Non conforme Document rejeté (qualité, format ou contenu incorrect) Soumettre un nouveau document corrigé

5.5 Document refusé — que faire ?

Si un document est marqué « Non conforme », la raison du refus est indiquée à côté du document. Voici les causes les plus courantes et les actions à effectuer :

Cause fréquenteSolution
Image floue ou illisibleRescanner ou rephotographier avec meilleure qualité
Document expiréObtenir un document à jour et le soumettre
Mauvaise traductionFaire traduire le document en chinois ou en anglais
Format incorrectConvertir au format demandé (ex. PDF)

5.6 Mises à jour en temps réel

Dès que l'équipe valide ou rejette un document, votre statut se met à jour automatiquement sans rafraîchissement de page.

7. Déclarer un Paiement

7.1 Principe

Lorsque vous avez effectué un paiement (virement bancaire, dépôt en agence, etc.), vous devez le déclarer dans le portail pour que l'équipe puisse le valider. Cette déclaration peut inclure une preuve de paiement (capture d'écran de virement, reçu bancaire, etc.).

7.2 Comment déclarer un paiement

Étapes :
  1. Allez dans la section « Paiements »
  2. Trouvez la tranche à payer (statut En attente)
  3. Cliquez sur « Déclarer ce paiement »
  4. Une fenêtre s'ouvre avec le formulaire de déclaration
  5. Vérifiez le montant affiché
  6. Indiquez la date à laquelle vous avez effectué le paiement
  7. Optionnellement, attachez une preuve de paiement (photo reçu, virement)
  8. Cliquez sur « Déclarer »
  9. Le statut passe à En validation — l'équipe est notifiée automatiquement

7.3 Après la déclaration

  • Le paiement passe au statut « En validation »
  • L'équipe Joda Company reçoit une notification
  • Un agent vérifie votre paiement et le valide (ou vous contacte si problème)
  • Une fois validé, le statut passe à « Payé » et un reçu est disponible
Délai de validation : En général, les paiements déclarés sont validés sous 24 à 48 heures ouvrables.

Pour toute question concernant votre dossier, vos paiements ou vos documents, vous pouvez contacter votre agent Joda Company :

  • Par téléphone (numéro fourni par votre agent)
  • Par email (adresse fournie par votre agent)
  • En personne à l'agence
